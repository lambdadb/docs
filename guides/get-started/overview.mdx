title: Overview
---

LambdaDB is a serverless-native database for building accurate, secure, and scalable AI applications.
Use LambdaDB to store and search unstructured data at scale without managing any infrastructure.

LambdaDB gives you everything you need for retrieval:

* Store text, embeddings, and their metadata
* Vector search
* Full-text search
* Hybrid search
* Metadata filtering

In addition, LambdaDB provides a set of convenient features to help you manage your data:

* Configurable project-level rate limiting
* Continuous backups and point-in-time recovery (PITR)
* Zero-copy collection clone in seconds regardless of collection size
* Zero operations due to its serverless-native architecture
